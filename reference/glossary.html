Glossary

The canonical vocabulary for this course. Definitions follow the HoTT Book; every lesson uses these terms in exactly this sense. Printable — keep it beside you.

Notation follows the HoTT Book: \(a : A\) reads “a is a term of type A”; \(a =_A b\) is the identity type; \(a \jdeq b\) is judgmental equality.

Type \(A : \UU\)
A primitive notion — not built from a prior notion of “set of elements.” Under the homotopy interpretation, a type is read as a space (and as an \(\infty\)-groupoid). \(\UU\) is a universe: a type whose terms are themselves types. [1, §1.3]
Term / element \(a : A\)
An inhabitant of a type. Under the interpretation, a point of the space \(A\). A term always comes with a type; “\(a\)” alone is meaningless. [1, §1.1]
Judgmental (definitional) equality \(a \jdeq b\)
Equality that holds by definition / computation. It is a judgment of the meta-theory, not a type, so it carries no evidence and cannot be reasoned about inside the theory. Decidable, “invisible.” [1, §1.1, §2 intro]
Identity type / propositional equality \(a =_A b\), also \(\mathsf{Id}_A(a,b)\)
The type whose terms are proofs that \(a\) and \(b\) are equal. A type family \(A \to A \to \UU\). Because it is a type, its inhabitants are first-class objects you can reason about, transport along, and ask further questions of. Under the interpretation, the path space from \(a\) to \(b\) in \(A\). [1, §1.12]
Path / identification \(p : a =_A b\)
A term of an identity type. Read as a path from \(a\) (its start point) to \(b\) (its end point) in the space \(A\). “Identification” is the logical reading; “path” the homotopical one — same object. [1, ch.2 intro]
Reflexivity \(\refl{a} : a =_A a\)
The canonical proof that \(a\) equals itself; the introduction rule for identity types. Under the interpretation, the constant path at \(a\). [1, §1.12]
Proof-relevance
The defining stance: a proof of \(a =_A b\) is data, and an identity type may have many distinct inhabitants — “just as there can be more than one path between two points of a space.” The break from set-level mathematics, where equality is a mere true/false proposition. [1, §1.12]
2-path / homotopy \(r : p =_{(a =_A b)} q\)
A path between parallel paths \(p, q : a =_A b\) — i.e. an inhabitant of an iterated identity type. The homotopical analogue of a homotopy rel endpoints. Iterating gives 3-paths, 4-paths, … [1, ch.2 intro]
\(\infty\)-groupoid
An algebraic structure of objects, morphisms between objects, morphisms between those, … with identity/composition/inverse at each level satisfying the groupoid laws up to higher morphisms (“weakly”). Every type carries this structure automatically via its tower of identity types. [1, ch.2 intro]
Path induction (\(\J\))
The elimination rule for identity types: to define something for all \(x,y\) and all \(p : x =_A y\), it suffices to handle the case \(x \jdeq y\) with \(p \jdeq \refl{x}\). Subtle but central — all higher-groupoid structure is derived from it. Slogan: the identity family is freely generated by reflexivity. [1, §1.12]
Based path induction
A convenient equivalent of path induction: fix \(a\); to define something for all \(x\) and \(p : a =_A x\), handle only \((a, \refl{a})\). Homotopically, the based path space at \(a\) is contractible. [1, §1.12]
Path inverse \(\opp{p} : y =_A x\)
The reversal of a path \(p : x =_A y\); symmetry of equality. Built by path induction (HoTT Book Lemma 2.1.1) with computation rule \(\opp{\refl{x}} \jdeq \refl{x}\). Not assumed — derived. [1, §2.1]
Concatenation / composite \(p \ct q : x =_A z\)
Composition of \(p : x =_A y\) with \(q : y =_A z\); transitivity of equality. Built by a double path induction (Lemma 2.1.2), with computation rule \(\refl{x} \ct \refl{x} \jdeq \refl{x}\). Notated in the opposite order from function composition. [1, §2.1]
Computation (conversion) rule
The judgmental equation an induction principle satisfies on its constructor. For \(\J\): \(f(x,x,\refl{x}) \jdeq c(x)\) — the value you specified on \(\refl{}\) really is the value taken there. The analogue of “the \(0^{\text{th}}\) term of a recursively defined sequence is the given one.” [1, §1.12]
Synthetic vs. analytic
Classical homotopy theory is analytic: spaces and paths are made of points (Cartesian coordinates, \(\mathbb{R}^n\)). HoTT is synthetic: points, paths, and higher paths are primitive, indivisible notions, governed by rules rather than constructed from a substrate. [1, ch.2 intro]
